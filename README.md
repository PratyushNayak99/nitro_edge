# EdgeSentinel

Secure, bare-metal AI incident-response copilot for Fedora Linux edge devices.

## What It Includes

- Strictly allowlisted diagnostics and system-service operations
- A cyberpunk incident-command widget for tool output
- Kernel-log resource access for investigations
- Studio-friendly development workflow

## Quick Start

```bash
cd edge-sentinel
npm run dev
```

## Common Commands

```bash
npm run dev
npm run build
npm start
npm run widget <command>
```
